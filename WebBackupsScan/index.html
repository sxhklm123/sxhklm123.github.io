WebBackupsScan-1.4 网站备份扫描软件

发表于 2019-02-19 21:57   文章字数 195


本工具开发于2017-1-25 21:02:57

主要用于对网站备份文件批量扫描

GET模式会对文件头标识符进行验证，扫到的100%都可以访问下载

使用.NET 2.0开发，使用前要确保电脑安装.NET 2.0环境

程序后续会继续更新功能上去。

字典变量使用

%wwwroot%.rar - http://www.baidu.com -> http://www.baidu.com/www.baidu.com.rar

%www%.rar - http://www.baidu.com -> http://www.baidu.com/baidu.rar

%root%.rar - http://www.baidu.com -> http://www.baidu.com/baidu.com.rar
